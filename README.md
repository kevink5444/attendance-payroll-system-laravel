# Attendance & Payroll System

## Description
Web-based system to manage employee attendance and payroll automatically.

## Features
- Login & authentication
- Employee management
- Attendance tracking
- Automatic payroll calculation

## Tech Stack
- Laravel
- MySQL
- Bootstrap

## Installation
git clone https://github.com/kevink5444/attendance-payroll-system-laravel.git
cd attendance-payroll-system-laravel
composer install
php artisan migrate
php artisan serve
